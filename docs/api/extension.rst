Serwer rozszerzenia
====================

Lokalny serwer Flask nasłuchujący na ``http://localhost:8765``. Udostępnia
aktywny profil i listę blokowanych stron dla rozszerzenia Chrome.

Szczegóły i przykłady zapytań: :doc:`../extension`.
